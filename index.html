Production-ready for aerospace inspection

Balloon drawings. Capture results. Export reports.

Axis is a Windows application for aerospace-quality inspection and drawing ballooning. Render manufacturing PDFs, place precision balloons, record measurements by work order/serial, and export ballooned drawings plus formatted inspection reports—backed by a single SQLite database per PDF.
