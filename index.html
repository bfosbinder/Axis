Key capabilities

Everything you need to balloon drawings, run inspection workflows, and export clean results—without juggling separate files.

Guided ballooning

Drag a rectangle on the drawing to create an auto-numbered balloon with stored page, bounds, zoom, offsets, and specs. Move balloons freely and resize in bulk.

High-fidelity PDF rendering

PyMuPDF provides crisp PDF drawing at varying zoom levels; a QGraphicsView overlay keeps balloons and highlights aligned to document coordinates.

Inspection workflows

Choose Ballooning or Inspection mode per session. Work orders/serials keep their own measurement data, and PASS/FAIL recalculates automatically for numeric or text inputs.

Reporting & exports

Export filtered inspection data to CSV, dump every work order to a master CSV, and generate both *_inspection.pdf and *_ballooned.pdf. Exports honor filters and status formatting.

Data integrity

All feature metadata and inspection results live in a single SQLite database next to the PDF, with automatic migration from legacy CSV files.

Tolerance parsing

Enter tolerance strings like 1.203 +/-.003 to auto-fill Nominal/LSL/USL and drive real-time status evaluation.

Product workflow

From PDF → balloons → measured results → exports, with everything saved per work order.

Data storage (per PDF)

Each drawing gets a sibling database: yourfile.axis.db.

features — feature rows matching the legacy master schema (page, bounds, zoom, method, specs, offsets, radius).

results — measurement values per (feature_id, workorder) with timestamps.

When the DB is created, Axis automatically imports legacy *.balloons.csv and <pdf>.<workorder>.csv it finds so existing projects remain intact.

Get Axis

Windows executable (preferred)
GitHub → Actions → “Build Windows EXE” → download the most recent artifact bundle → extract and run AxisInspector.exe (no install, no Python required).

Development build (Python)

python3 -m venv .venv
source .venv/bin/activate        # use .venv\Scripts\activate on Windows
pip install -r requirements.txt
python main.py
